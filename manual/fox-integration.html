FOX toolkit integration¶

The Libcanvas library can be used inside a FOX Toolkit application using a special class FX::FXElpWindow declared in the header file FXElpWindow.h.

The FX::FXElpWindow inherit from FX::FXWindow and can be used as an ordinary FOX Window in every respect. In addition the class implements the method FX::FXElpWindow::Attach() to bind a plot, like it can be done with a elp::Window.

The plots themselves can be created and modified using the standard Libcanvas functions.

Once a plot is bound to a window the application will be automatically updated for any plot’s modification, without the need of any explicit action.
